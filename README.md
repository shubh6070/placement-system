A web-based application developed using **PHP & MySQL** that helps manage internship and placement activities efficiently for students and administrators.

---

## 📌 Overview

The **Placement & Internship Management System** is designed to provide a centralized platform where:

- 🎓 Students can explore and apply for jobs
- 👨‍💼 Admin can manage companies, jobs, and applications

This system simplifies the recruitment process and improves communication between students and administrators.

---

## 🎯 Features

### 👨‍🎓 Student Module
- Student Registration & Login
- View Internships & Placements
- Search & Filter Opportunities
- Apply for Jobs
- Track Application Status
- Update Profile (Skills, Education, Resume, Image)

### 👨‍💼 Admin Module
- Admin Login (Separate Panel)
- Add Companies
- Add Internship & Placement Jobs
- View Applications
- Select / Reject Candidates
- Dashboard with Statistics
